## 주요기능

* React JS를 이용한 Twitter 기능 구현
* Firebase API 를 이용한 다양한 코드 구현
* FileReader API 를 이용한 image Reader 기능 구현
* Hook(useState, useEffect, useHistory etc..)을 활용한 코드 생성
* 모바일 친화적 화면 구현
